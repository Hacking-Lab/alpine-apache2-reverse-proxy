# Backend Service
This content is delivered by the `backend` service behind the Apache reverse proxy.

```
            +------------+                +---------------+
            | Apache     |                | Apache + PHP  |
            | Reverse    |                | Backend       |
  +-------->+ Proxy      +--------------->+ Service       |
            |            |                |               |
            +------------+                +---------------+

```
